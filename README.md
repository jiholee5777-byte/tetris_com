# Season Tetris 🌸❄️🍂☀️

**Season Tetris**는 전통 테트리스의 기본 재미에 **계절 변화와 시각적 효과**를 결합한 브라우저용 HTML5 게임입니다.  
플레이어는 떨어지는 테트리스 블록을 회전하고 이동시켜 한 줄씩 완성하며 점수를 쌓습니다.  
게임 화면은 계절과 점수에 따라 시시각각 변하며, **눈에 띄는 색감과 움직임**으로 몰입감을 높입니다.

---

## 🎨 게임 스타일

- **계절별 시각 테마**
  - **봄**: 연한 초록 배경과 파스텔 톤 블록, 산뜻한 분위기
  - **여름**: 청록색 배경과 밝은 원색 블록, 시원하고 활기찬 느낌
  - **가을**: 연갈색 배경과 따뜻한 오렌지·갈색 계열 블록, 포근한 분위기
  - **겨울**: 연회색 배경과 하늘·흰색 계열 블록, 차분하고 깨끗한 느낌
- **라인 제거 효과**
  - 한 줄을 없앨 때 화면이 살짝 흔들리면서 **파워풀한 피드백** 제공
  - 줄 제거 시마다 **점수가 +10** 증가, 화면 오른쪽 위 점수 표시
- **블록 디자인**
  - 클래식 테트리스 블록 형태
  - 계절별 색상 테마 적용
  - 블록 움직임과 회전이 부드럽게 느껴지도록 설계
- **게임 속도**
  - 초반 느린 속도에서 시작하여, 플레이어의 점수와 상관없이 안정적인 템포 유지
  - 즉시 드롭, 빠른 하강 등 직관적인 컨트롤 지원
- **점수 기반 계절 변화**
  - **20점 단위로 배경과 블록 색상 변경**
  - 플레이어는 점수를 쌓으면서 시각적 변화를 경험

## 🎮 조작 방법

| 키 | 동작 |
|----|------|
| ←  | 블록 왼쪽 이동 |
| →  | 블록 오른쪽 이동 |
| ↓  | 블록 아래로 빠르게 이동 |
| ↑  | 블록 회전 |
| Space | 블록 즉시 드롭 |

---

## 🏆 플레이 감각

- **몰입감 있는 시각 변화**: 점수를 올릴 때마다 배경과 블록 색이 바뀌어 계절감을 직관적으로 체감
- **피드백 중심 디자인**: 줄 제거 시 흔들림과 점수 증가로 성취감 강화
- **깔끔하고 심플한 UI**: 점수 표시 외 불필요한 인터페이스 제거, 오로지 블록과 배경에 집중
- **경쾌한 반복성**: 전통 테트리스의 리듬감과 계절별 색상 변화를 통한 변주감이 결합
